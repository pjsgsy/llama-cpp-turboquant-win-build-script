Full build script for my Win11/VisualStudio/CUDA12.9 setup

Will fetch / update source directly from Tom's GitHub, and compile, so it is an all-in-one build from nothing script.
Should be able to just run it and update at any time when the author updates his source.
It will possibly take an hour or more for the first build, if you are hardware-constrained, like me! (a good time to have a good coffee supply...)
